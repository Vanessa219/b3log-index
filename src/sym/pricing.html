如何使用免费的社区版？

  • 下载 GitHub 开源代码，参照安装指南进行搭建。使用时请注意保留底部版权信息及相关链接： © 2012-present B3log 开源 • Sym

商业版不同价格的区别？

  • 对于非营利性组织（需提供社会团体登记证明）提供五折优惠，即 10000 RMB
  • 对于工商注册一年内的初创公司提供八折优惠，即 16000 RMB
  • 对于常规类型公司实体价格为 20000 RMB
  • 除价格外其他都相同，采购源码后可终身使用

社区版、商业版及定制版的区别？

  • 商业版在功能、性能、移动端、界面以及安全等多方面做了更细致的优化。支持多节点集群部署；提供专业技术支持， 具体可参见 Sym 商业版优势
  • 定制版是在商业版基础上提供的二次开发服务，按需求评估工时进行报价，1000 RMB / 人天，驻场开发价格请咨询客服

所有主题都免费吗？

  • 目前内置的主题有 Classic、Mobile
  • 除内置主题外，其余主题 5000 RMB / 套
  • 可前往链滴设置体验

采购流程？

  • 甲方填写合同后打印、盖章并邮寄

  • 乙方发送源码并提供技术支持或定制开发服务
